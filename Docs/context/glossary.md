# Glossary

- Slot: intervallo temporale candidato per un appuntamento
- Resource: entità schedulabile (medico, stanza, macchinario)
- Busy interval: intervallo occupato da prenotazione
- Negative event: intervallo reso indisponibile da blocchi/ferie/manutenzione
- Normalization: trasformare eventi di dominio in input semplice per il core
- Hot-path: parte del codice eseguita frequentemente e sensibile a performance
